# vision-hud-controller

Computer vision demo with a simple HUD/overlay and adjustable settings.

## Quickstart
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

python -m src.main --source 0
```

## Controls
- `q` quit
- `+` increase HUD opacity
- `-` decrease HUD opacity
- `s` save settings
